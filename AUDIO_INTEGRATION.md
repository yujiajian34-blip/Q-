## 音效播放方式

1. **PyAudio** (首选)
   - 需要安装: `pip install PyAudio`

   - 状态：未安装时自动跳过

2. **Windows winsound** (备选)
   - 内置Windows功能
   - 优点：无需安装
   - 状态：✅ **当前使用中**

## 如果没有声音

### 排查步骤

1. **检查文件**
   ```bash
   Get-ChildItem "wav" -Filter "*.wav" | Measure-Object
   ```
   应该显示: **13**

2. **检查系统音量**
   - Windows 任务栏音量控制是否在静音？
   - 应用是否被系统静音？

3. **验证音效格式**
   ```bash
   python -c "import wave; w = wave.open('wav/熊家肚子饿饿的.wav'); print(w.getparams())"
   ```
   应该显示音频参数而不出错

4. **直接测试 winsound**
   ```bash
   python -c "import winsound; winsound.PlaySound('wav/熊家肚子饿饿的.wav', winsound.SND_FILENAME)"
   ```
   这应该播放音效或没有错误

5. **安装 PyAudio (可选)**
   ```bash
   pip install PyAudio
   ```
   这样可以使用更完善的音效播放
