## 📦 新增模块详解

### 1️⃣ `bubble_manager.py` - 思维气泡管理系统
**功能**：管理所有思维气泡的生成、触发和去重

**主要特性**：
- ✅ 8 大主题气泡内容库（100+ 条内容）
- ✅ 自适应触发（根据时间、心情、行为调整）
- ✅ 智能去重（45 秒间隔限制 + 2 小时内容去重）
- ✅ 根据文本长度自动调整显示时间
- ✅ 时段自适应（早上/深夜倾向显示睡眠相关）

**关键方法**:
```python
get_bubble(theme=None)                # 获取气泡
get_contextual_bubble(action)         # 根据行为获取相关气泡
get_bubble_duration(text)             # 计算显示时长
clear_history()                       # 清除历史记录
```

### 2️⃣ `diary_generator.py` - 日记生成引擎
**功能**：基于模板和实时数据自动生成每日便签日记

**主要特性**：
- ✅ 开头、中间、结尾模板自由组合
- ✅ 根据互动次数自动调整语气
- ✅ 根据心情值生成相应内容
- ✅ 时段感知（根据时间生成内容）
- ✅ 自动去重（防止短期重复）

**生成流程**:
```
统计当天数据 (心情、互动、行为)
        ↓
选择合适的模板组合
        ↓
随机填充动态词汇
        ↓
生成 80-200 字日记
        ↓
自动保存到 JSON
```

**关键方法**:
```python
generate_daily_diary(stats)           # 生成今日日记
get_diary_by_date(date)              # 读取特定日期日记
get_all_diaries()                    # 获取所有日记
get_statistics()                     # 统计信息
```

### 气泡触发规则

```
显示触发：
  • IDLE 状态：90-240 秒有 30% 概率
  • 行为后：20% 概率冒出相关主题气泡
  • 交互后：15% 概率冒出反应气泡

限制规则：
  • 两次间隔 ≥ 45 秒
  • 同内容 2 小时内不重复
  • 语音播放时不显示
  • SLEEPING 状态不显示
  • 深夜不显示
```

## 💾 数据存储结构

### 日记保存格式（pet_diary.json）

```json
{
  "diaries": [
    {
      "date": "2026-04-15",
      "content": "嘿嘿，熊家又来写日记了...",
      "stats": {
        "clicks": 12,
        "drags": 3,
        "throws": 1,
        "mood": 85,
        "day_count": 15,
        "behaviors_played": 8
      },
      "created_at": "2026-04-15T22:00:00"
    }
  ]
}
```

### 防重复机制

**三层防重复**：
1. **模板去重**：同模板 5 天内不重复
2. **组合去重**：同组合 3 天内不重复
3. **相似度检测**：与最近 5 篇相似度 > 70% 时重新生成

**理论不重复天数**：
```
12开头 × C(40,2.5)中间 × 10结尾 × 10天气 × 10心情 ≈ 数千种组合
```

## 💡 设计亮点

### 1. 智能去重机制
- 算法层面去重（45 秒间隔）
- 内容层面去重（相似度检测）
- 权重调整（某些主题白天显示频率低）

### 2. 动态内容生成
- 不是硬编码，而是模板 + 数据驱动
- 同一模板 + 不同数据 = 完全不同的日记
- 理论上可以生成数千种不同日记

### 3. 用户体验优先
- 气泡不打扰工作（自动消失、间隔长）
- 日记查看方便（日历 + 快捷导航）
- 数据累积价值（记录每天的故事）

### 4. 高可扩展性
- 模板易于扩展（只需添加字符串）
- 主题易于添加（enum 扩展）
- 词库易于丰富（列表扩展）
