## 🐛 常见问题

### ❌ 点击菜单后没有反应
**解决方案：**
1. 确保程序已完全启动（看到宠物出现在屏幕上）
2. 尝试重新启动程序
3. 检查是否有其他窗口在后台

### ❌ 日记窗口闪现后关闭
**可能原因：** 日记窗口初始化出错
**解决方案：**
1. 运行 `python test_diary.py` 检查
2. 查看控制台是否有错误信息

### ❌ 看不到之前的日记
**原因：** 日记数据存储在 `pet_diary.json`
**检查：**
1. 确认文件存在：`ls pet_diary.json`
2. 文件损坏时可删除重建（会失去之前的日记）

## 🔍 诊断步骤

如果日记仍无法打开，请按顺序执行：

### 第一步：验证功能
```bash
python test_diary.py
```
应该看到所有测试都是 ✅ 通过

### 第二步：检查日记文件
```bash
dir pet_diary.json  # Windows
# 或
ls -la pet_diary.json  # Mac/Linux
```
应该能看到文件存在

### 第三步：检查程序日志
启动程序时打开终端查看是否有错误：
```bash
python main.py
# 观察控制台输出
```
