# 💭 Q宠 思维气泡与日记系统 - 使用指南

## 🎉 新功能简介

你的熊家宠物现在拥有更丰富的表达力！

### ✨ 新增功能

#### 1️⃣ **思维气泡系统**
- 📌 熊家会在闲置时随机冒出想法气泡
- 💭 每条气泡都代表它当下的某个想法
- 🎨 语气可爱、会自动消失、不打扰工作
- 🔄 两条气泡之间至少相隔 45 秒
- 📚 90+ 条不同主题的想法内容

#### 2️⃣ **熊家日记本**（全新便签风格）
- 📖 每天自动生成一篇独特的日记
- 📅 日历 + 便签双视图
- 📊 显示每日互动数据统计
- 🎯 内容基于当天的互动、心情、时间等自动组合生成
- 🔍 支持翻阅历史日记

### 思维气泡触发规则

```
主动触发：
  • IDLE 状态 → 每 90-240 秒有 30% 概率冒出
  • 行为后触发 → 20% 概率冒出相关主题的气泡
  • 用户交互后 → 15% 概率冒出反应气泡

显示规则：
  ✓ 气泡间隔 ≥ 45 秒
  ✓ 同一内容 2 小时内不重复
  ✓ 语音播放时不显示气泡
  ✓ 深夜或睡眠状态不显示气泡
```

### 日记系统工作流程

```
每天 22:00（或程序启动时）
      ↓
收集当天数据：
  • 互动次数（点击、拖拽、甩飞）
  • 心情值变化
  • 使用天数
  • 演过的动作种类
      ↓
从 100+ 模板中智能组合
      ↓
自动生成 80-200 字的日记
      ↓
保存到 json 文件 + UI 展示
```

## 📚 如何使用

### 访问思维气泡

**自动触发** - 不需要做任何事情！
- 熊家会在闲置时自动冒出想法
- 气泡会自动消失，不需要手动关闭

### 访问日记本

**右键菜单** → **📖 查看熊家的日记本**

日记本窗口功能：
- 📅 **日历视图**：点击日期查看对应日记
- 📝 **日记内容**：阅读熊家那天的小日记
- ◀ / ▶ **翻页按钮**：查看前一天/后一天
- 📌 **今天按钮**：快速返回今天
- 📊 **统计信息**：显示该天的互动数据

## 💡 使用建议

### 最佳体验

1. **定期互动**
   - 经常点击、拖拽熊家
   - 执行不同的动作
   - 这样日记内容会更丰富多彩！

2. **定期查看日记**
   - 每天都有新的日记！
   - 回顾最近的互动记录
   - 发现熊家的小秘密 😊

3. **注意时段**
   - 早上和周末的气泡/日记会更活泼
   - 深夜会比较安静
   - 这样更符合生活规律呢！

### 有趣发现

✨ **彩蛋**：
- 连续 7 天有日记 → 特殊气泡提醒
- 查看日记时 → 熊家会"害羞"的反应气泡
- 使用天数达到里程碑 → 特殊纪念日记

## 🆘 常见问题

**Q: 为什么没有看到气泡？**  
A: 
- 检查是否在 IDLE 状态（正在播放动画时不会出现）
- 如果刚看过气泡，需要等待 45 秒以上
- 深夜时段 (23:00-7:00) 概率较低

**Q: 日记内容为什么是重复的？**  
A: 系统设计了防重复机制，但 100+ 模板组合理论上几年才会重复。如果频繁看到重复，请反馈！

**Q: 如何手动触发日记生成？**  
A: 日记会在程序自动生成。如要查看今天，打开日记本窗口后点"📌 今天"按钮即可。

**Q: 气泡会打扰工作吗？**  
A: 不会！气泡会自动消失，而且间隔很长（90-240 秒）。
