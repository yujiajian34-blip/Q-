## 故障排查

### 如果仍然没有声音：

#### 1. 检查Windows音量
- 右下角音量图标
- 确保未静音
- 检查应用级别音量

#### 2. 重启应用
```bash
python main.py
```

#### 3. 验证文件
```bash
# PowerShell
Get-ChildItem "wav" -Filter "*.wav"
```
应该显示13个文件

#### 4. 直接测试
```bash
python -c "import winsound; winsound.PlaySound('wav/熊家肚子饿饿的.wav', winsound.SND_FILENAME)"
```
应该能听到"熊家肚子饿饿的"

#### 5. 详细诊断
```bash
python test_audio_playback.py
```
