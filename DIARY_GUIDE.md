# 📖 Q宠桌面宠物 - 日记功能使用指南

## 功能介绍

✨ **新增日记功能**：让你能够记录与布布的每日互动和心情变化！

### 核心功能

1. **📝 每日日记**
   - 记录每天与布布的点点滴滴
   - 支持查看和编辑历史日记
   - 自动保存日记内容

2. **😊 心情记录**
   - 开心 😊
   - 普通 😐
   - 难过 😢
   - 生气 😠

3. **📅 日期管理**
   - 日历选择任意日期
   - 查看最近30天的日记
   - 快速浏览日记列表

4. **✨ 事件自动记录**
   - 喂食：自动记录食物类型和效果
   - 互动：记录所有互动行为
   - 照顾：洗澡、唱歌、运动等行为都会被记录

## 使用方法

### 打开日记窗口

1. **右键点击宠物**
2. **选择 "📖 查看日记"** 选项
3. 日记窗口会打开

### 日记窗口说明

```
📍 左侧面板：
├── 📅 日期选择 - 用日历选择要查看的日期
└── 📝 日记列表 - 显示最近30天的所有日记

📍 右侧面板：
├── 📅 日期标题 - 显示当前选中的日期
├── 😊 心情选择 - 选择今天的心情
├── ✨ 事件列表 - 今天发生的所有事件
├── 📖 日记内容 - 编写或查看日记
└── 💾 保存、删除、关闭按钮
```

### 编写日记

1. **选择日期**
   - 使用左侧日历选择日期
   - 或从日记列表中点击

2. **选择心情**
   - 在"今天的心情"下拉框选择

3. **查看事件**
   - 自动显示该日期的所有互动事件
   - 包括：喂食、洗澡、唱歌、运动、互动等

4. **编写内容**
   - 在"日记内容"文本框中输入
   - 记录与布布的互动故事

5. **保存**
   - 点击"💾 保存日记"按钮保存

## 数据存储

- **日记保存位置**：`pet_diary.json`
- **保存格式**：JSON 格式，人类可读
- **自动保存**：每次编辑后立即保存
- **持久存储**：关闭程序后数据自动保留

## 常见问题

**Q: 日记会不会丢失？**
A: 不会。所有日记都自动保存在 `pet_diary.json` 中，即使重启程序也会保留。

**Q: 可以删除日记吗？**
A: 可以。打开要删除的日记，点击"🗑️ 删除日记"按钮确认即可。

**Q: 事件列表是自动生成的吗？**
A: 是的。当你与宠物互动（喂食、洗澡等）时，系统会自动记录。

**Q: 可以编辑过去的日记吗？**
A: 可以。从日历或列表中选择任何日期，即可编辑和保存该日期的日记。
