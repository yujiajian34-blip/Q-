## 🔧 自定义配置

### 修改气泡间隔
在 `bubble_manager.py` 修改：
```python
self.min_interval = 45    # 两次气泡间隔（秒）
```

### 修改日记生成时间
在 `diary_generator.py` 修改：
```python
daily_generation_hour = 22  # 每天 22:00 生成
```

### 添加自己的气泡内容
在 `bubble_manager.py` 的 `_init_bubble_pool()` 方法中添加：
```python
BubbleTheme.FOOD: [
    # ... 现有内容 ...,
    "你的新想法！"
]
```

## 🐛 常见问题

**Q: 为什么总是看不到气泡？**  
A: 检查：
1. 程序是否还在运行
2. 是否处于 IDLE 状态（没有播放动画）
3. 是否等待了足够的时间（90-240 秒）
4. 深夜时段（23:00-7:00）概率较低

**Q: 日记怎么都是重复的？**  
A: 系统有防重复机制，但 100+ 模板理论上数年才会重复。如果频繁见到，请反馈！

**Q: 如何查看历史日记？**  
A: 打开日记本 → 用日历点击历史日期 → 自动加载

**Q: 日记数据会丢失吗？**  
A: 不会！所有日记保存在 `pet_diary.json`，重启程序也保留。

**Q: 能否关闭气泡系统？**  
A: 可以，在 `pet_widget.py` 中注释掉 `self.thinking_bubble_timer` 即可。
