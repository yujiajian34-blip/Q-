# Q宠桌面宠物 🐱

一个可爱的桌面宠物程序，让你的桌面更加生动有趣！

## 功能特色

- 🎨 **透明窗口** - 完美融入桌面背景
- 🖱️ **拖拽互动** - 按住左键拖动，双击触发动作
- 🍴 **喂食系统** - 3种食物，不同效果
- 💬 **气泡对话** - 宠物会随机说话表达心情
- 😊 **心情系统** - 开心/普通/难过/生气四种状态
- 📌 **边缘吸附** - 自动吸附到屏幕边缘
- 💾 **数据保存** - 自动保存亲密度和心情数据

## 使用方法

### 方式一：直接运行 EXE（推荐）

1. 双击 `Q宠桌面宠物.exe` 即可运行
2. 右键点击宠物打开菜单

### 方式二：从源码运行

```bash
# 安装依赖
pip install -r requirements.txt

# 运行程序
python main.py
```

## 互动操作

| 操作 | 效果 |
|------|------|
| 左键拖动 | 移动宠物位置 |
| 双击 | 触发随机动作 |
| 右键菜单 | 打开功能菜单 |

## 菜单功能

| 功能 | 效果 |
|------|------|
| 🍴 喂食 | 喂西瓜/拉面/奶茶，增加亲密度和心情 |
| 🛁 洗澡 | 心情+10 |
| 🎵 唱歌 | 心情+5，亲密度+3 |
| 💃 运动 | 心情+8，亲密度+2 |
| 🌞 问候 | 心情+5，亲密度+5 |
| 🤝 互动 | 心情+10，亲密度+8 |
| 💕 表情 | 心情+5，亲密度+5 |
| 🐻 和布布玩 | 心情+8，亲密度+6 |
| 👊 欺负布布 | 心情-5，亲密度-3 |

## 打包说明

运行 `build.bat` 即可自动打包成 EXE 文件。

## 系统要求

- Windows 10/11
- 无需安装 Python 环境（EXE版本）
