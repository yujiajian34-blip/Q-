# Q宠 - 智能桌面宠物伴侣

![Python](https://img.shields.io/badge/Python-3.12-blue)
![PyQt5](https://img.shields.io/badge/PyQt5-5.15.11-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

一个功能丰富的Windows桌面宠物应用，融合了AI日记生成、心情追踪和互动系统。让您的桌面充满生动的陪伴与乐趣。

## ✨ 核心功能

### 🐻 虚拟宠物系统
- **40+个独特动画**：包括基础动画、交互动画、表情动画等
- **完整的生命模拟**：饥饿度、心情值、亲密度实时跟踪
- **多种互动方式**：
  - 🍴 喂食（10种不同食物）
  - 🛁 洗澡（恢复心情）
  - 🎵 唱歌（增加亲密度）
  - 💃 运动（释放压力）
  - 🤝 互动（深化关系）
  - 👊 欺负（特色交互）

### 💭 说说心里话（心情日记）
用户可以记录日常想法和心情：
- **8种心情标签**：开心、普通、难过、感慨、感谢、甜蜜、期待、伤心
- **智能日记参考**：宠物日记会根据用户心情智能回应
- **历史记录查询**：
  - 📅 日期导航（支持前后翻页）
  - 🗓️ 日历选择（快速跳转任意日期）
  - 📖 历史浏览（回顾过往心情）
- **50%概率引用**：宠物日记中随机引用用户原始想法

### 📖 熊家日记本
- **每日日记生成**：根据宠物状态和用户心情自动生成独特日记
- **日记分析**：查看历史日记，追踪宠物成长故事
- **心情统计**：可视化的情感数据分析
- **日签内容**：每日特色温暖文案

### 🎨 丰富的视觉系统
- **思维气泡**：宠物随机显示内心想法（90-240秒间隔）
- **色彩情感**：根据心情改变宠物色调
- **流畅动画**：WebP格式确保清晰流畅的视觉效果
- **桌面适配**：自动贴边吸附到屏幕边缘

### 🔊 音频系统
- **多场景音效**：
  - 喂食满足音
  - 心情变化音
  - 互动快乐音
  - 背景音乐
- **Windows音效API**：自动适配系统音频

### ⚙️ 高级功能
- **开机自启**：通过Windows注册表设置自动启动
- **置顶显示**：保持在其他窗口之上
- **数据持久化**：JSON格式保存所有数据
- **自动保存**：定时保存用户数据，不怕意外关闭

## 📦 安装部署

### 方式一：直接运行EXE（推荐）
1. 下载 `dist/Q宠.exe` 文件（41.39 MB）
2. 双击运行即可
3. 所有依赖和资源已打包在内

### 方式二：Python源码运行
**系统要求**：
- Windows 7 及以上
- Python 3.12+

**安装依赖**：
```bash
pip install -r requirements.txt
```

**运行程序**：
```bash
python main.py
```

## 🌟 核心算法

### 三层心里话参考机制
1. **第一层**：获取今天用户的所有心里话记录
2. **第二层**：根据心情标签查找预设的回应模板
3. **第三层**：50%概率随机引用用户原始想法

### 心情值变化规则
| 操作 | 心情变化 | 亲密度变化 |
|------|--------|----------|
| 喂食 | -5 | +0 |
| 洗澡 | +8~12 | +0 |
| 唱歌 | +3~7 | +1~5 |
| 运动 | +6~10 | +0~4 |
| 问候 | +3~7 | +3~7 |
| 互动 | +8~12 | +6~10 |

## 🐛 故障排除

### exe运行报错
- **解决**：确保Windows 7及以上版本
- **检查**：.Net Framework 4.5+ 已安装

### 心里话窗口无响应
- **解决**：检查`inner_thoughts.json`文件权限
- **方案**：删除该文件，重新启动程序

### 音效不播放
- **原因**：PyAudio未安装（已自动降级到Windows API）
- **状态**：程序会自动使用Windows音效系统

### 日记无法生成
- **检查**：`pet_diary.json`文件是否可写
- **方案**：确认目录权限正确
